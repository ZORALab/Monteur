Job Level Variables Section

The Monteur's Variables Processing features. All variables defined in this job-wide configuration file only visible to ALL {{ .JobTitle }} CI Job recipes. It is highly recommended to place all project-based variables data into this list for recipes consistency purposes.

An example of the job-level variables section consists of the following fields: {{ .SampleCodeBlock }}

  • [Variables] - plain variables list complying to Montuer Plain Variables Definition.

  • [FMTVariables] - formattable variables list complying to Montuer Formattable Variables Definition.
